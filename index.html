General Information

This workshop starts with basic information about R syntax, the RStudio interface, and move through how to import CSV files, the structure of data frames, how to deal with factors, how to add/remove rows and columns, how to calculate summary statistics from a data frame, and a brief introduction to plotting. The lesson can be found here.

This hands-on one-day workshop is based on Data Carpentry's R lesson. Software Carpentry and Data Carpentry's mission is to help scientists and engineers get more research done in less time and with less pain by teaching them basic lab skills for scientific computing.

Who: This workshop is intented for participants with no programing experience but an interest in learning R.
